FORMULÁRIO

O formulário é a tag < form> que é utilizada em muitos sites para ter um contato maior entre desenvolvedor e usuário.

Dentro de form tem a tag < input> na qual é criada os elementos de um formulário, como botões e caixas de texto

Para escolher o tipo do seu input, deve lançar o atributo type="" Dento da tag input para escolher.

Por exemplo, se for um input de texto, só por type="text" que vai aparecer uma caixa de texto, caso queira uma caixa de texto para senha, só por type="password" que iria alterar o padrão da caixa.

Caso queira adicionar um botão, só por type="button" value="Algo que vc queira escrito dentro do botão"

Se o caso for um texto grande, usa-se a tag < textarea> para coloca-lo.
